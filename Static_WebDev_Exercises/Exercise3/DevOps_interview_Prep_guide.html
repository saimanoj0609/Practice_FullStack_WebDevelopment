  • Docker basics and Dockerfiles
  • Kubernetes core components and YAML

  • Git Basics: branches, merges, rebases

  • Conflict resolution and Git hooks

  • Describe a CI/CD failure you resolved
  • How do you handle production incidents?
  • Tell me about a time you automated a manual process
